Итерация 2:
1. Депозит денег пользователем
2. Перевод денег с одного аккаунта на другой
3. Изменение имени пользователя

Требования:
1) Имя профиля - два слова сотсоящие только из букв обязательно
2) Максимальная сумма депозита 5000
3) Максимальная сумма перевода 10000

# Депозит денег пользователем
## Позитив: пополнение счета валидной суммой 5000
Предусловие: зарегистрировать пользователя и создать счет на профиль пользователя. Баланс на счету 0.
1. Авторизоваться по логину паролю
2. Нажать на кнопку Deposit Money
3. В дропдаун меню Select Account выбрать существующий счет
4. В поле ввода Enter Amount: ввести валидную сумму пополнения 5000
5. Нажать кнопку Deposit
   ОР: Появился алерт "✅ Successfully deposited $1000 to account ACC2!", на бэкэнде по вызову ручки /api/v1/auth/login сумма пополнилась на указанную в пополнении сумму

## Негатив: пополнение счета суммой превышающей валидный диапазон чисел 1000000
Предусловие: зарегистрировать пользователя и создать счет на профиль пользователя. Баланс на счету 0.
1. Авторизоваться по логину паролю
2. Нажать на кнопку Deposit Money
3. В дропдаун меню Select Account выбрать существующий счет
4. В поле ввода Enter Amount: ввести сумму пополнения 1000000
5. Нажать кнопку Deposit
   ОР: Появился алерт "❌ Please deposit less or equal to 5000$.", на бэкэнде по вызову ручки /api/v1/auth/login сумма не изменилась (==0)


# Перевод денег с одного аккаунта на другой
## Позитив: перевод валидной суммы 10000 между своими счетами одного пользователя
Предусловие: зарегистрировать пользователя, установить валидное значение name и создать 2 счета на профиль пользователя. Пополнить баланс на первом счету пользователя на 10000.
1. Авторизоваться по логину паролю
2. Нажать на кнопку Make a Transfer
3. В дропдаун меню Select Your Account выбрать первый счет (с балансом 10000)
4. В поле ввода Recipient Name ввести name (не username) самого пользователя
5. В поле ввода Recipient Account Number ввести номер второго счета самого пользователя
6. В поле ввода Amount указать 10000
7. Клик чекбок Confirm details are correct
8. Нажать кнопку Send Transfer
   ОР: Появился алерт "✅ Successfully transferred $1 to account ACC2!", на бэкэнде по вызову ручки /api/v1/auth/login на счету 1 баланс уменьшился на сумму перевода, на счету 2 баланс увеличился на сумму перевода

## Негатив: перевод невалидной суммы 10000.01 между своими счетами одного пользователя
Предусловие: зарегистрировать пользователя, установить валидное значение name и создать 2 счета на профиль пользователя. Пополнить баланс на первом счету пользователя на 10000.01.
1. Авторизоваться по логину паролю
2. Нажать на кнопку Make a Transfer
3. В дропдаун меню Select Your Account выбрать первый счет (с балансом 10000.01)
4. В поле ввода Recipient Name ввести name (не username) самого пользователя
5. В поле ввода Recipient Account Number ввести номер второго счета самого пользователя
6. В поле ввода Amount указать 10000.01
7. Клик чекбок Confirm details are correct
8. Нажать кнопку Send Transfer
   ОР: Появился алерт "❌ Error: Transfer amount cannot exceed 10000", на бэкэнде по вызову ручки /api/v1/auth/login на счету 1 баланс не изменился, на счету 2 баланс не изменился

# Изменение имени пользователя
## Позитив: установка валиадного имени
1. Авторизоваться по логину паролю
2. В верхнем правом углу рядом с Logout нажать на название своего профиля
3. В поле ввода Edit Profile ввести текст John Doe
4. Нажать на кнопку Save changes
   ОР: Появился алерт "✅ Name updated successfully!", на бэкэнде по вызову ручки /api/v1/customer/profile name в профиле изменился на указанное значение

## Негатив: установка невалидного имени два слова с цифрами
1. Авторизоваться по логину паролю
2. В верхнем правом углу рядом с Logout нажать на название своего профиля
3. В поле ввода Edit Profile ввести текст John Doe111
4. Нажать на кнопку Save changes
   ОР: Появился алерт "Name must contain two words with letters only", на бэкэнде по вызову ручки /api/v1/customer/profile name в профиле не изменился на указанное значение
